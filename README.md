# MBAD Final Project
Ứng dụng mobile xây dựng bằng Flutter giúp theo dõi dữ liệu cảm biến (nhiệt độ, độ ẩm) theo thời gian thực.  
Dữ liệu được lấy từ Firebase và hiển thị trực quan qua dashboard và biểu đồ.

## Công nghệ sử dụng
- Flutter (Dart)  
- Firebase Realtime Database  
- REST API  
- Thư viện biểu đồ  

## Tính năng chính
- Theo dõi dữ liệu cảm biến theo thời gian thực  
- Hiển thị dữ liệu dưới dạng biểu đồ  
- Hỗ trợ Android, Web
